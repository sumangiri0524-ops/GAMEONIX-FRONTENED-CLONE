# 🎮 GAMEONIX FRONTEND CLONE

A modern **Gaming Platform  Clone** inspired by **GAMEONIX**.
This project recreates a professional esports platform UI using **HTML, CSS, and JavaScript** with advanced animations and interactive components.

The website showcases gaming tournaments, popular games, downloads, FAQ, and support sections with smooth animations using **GSAP** and **Swiper.js**.

---

# 🚀 Live Features

✨ Fully Responsive Design
✨ Animated Hero Section
✨ Floating Bubble Background Animation
✨ Swiper Slider for Gaming Promotions
✨ Interactive Game Cards with Hover Animation
✨ Scroll Animations using GSAP + ScrollTrigger
✨ FAQ Accordion System
✨ Responsive Mobile Navigation Menu
✨ Support & Helpdesk Section

---

# 🛠️ Technologies Used

* **HTML5**
* **CSS3**
* **JavaScript (Vanilla JS)**
* **GSAP (GreenSock Animation Platform)**
* **GSAP ScrollTrigger**
* **Swiper.js**
* **Remix Icons**
